# Capas

1. floor and border: capa con diseño del suelo y exteriores.
2. walls: capa con paredes del laberinto, las cuales deberás limitar el movimiento del personaje en el juego.
3. trivia_points: capa con los 10puntos de trivia repartidos por el mapa. Abrirán una pregunta con opciones de respuesta.
4. traps: capa con obstáculos a ser evadidos mientras se recorre el mapa.
5. end: capa con el punto final del laberinto para detectar su fin.
